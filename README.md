# CS3300 Group Project - VibeVision
by Jared Abels, Syndeed Ayman, Lydia Easter, Joshua Leisman 

Team "CS is just our hobby"

Intro to Software Engineering Group Project - Android App interface for ML mood model

## Problem Statement:
We plan to develop an Android application that uses a pre-existing machine learning model to 
determine whether a user appears happy or sad based on an image captured through the device’s camera.
The app will process facial expressions in real time, classify emotional states using a trained 
image‑based ML classifier, and provide feedback within the user interface.​
